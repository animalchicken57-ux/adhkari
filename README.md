# أذكاري — متتبّع أذكار الصباح والمساء 🕌

تطبيق ويب يساعدك على المحافظة على **أذكار الصباح والمساء** عبر عدّاد تفاعلي،
متابعة يومية، وسلاسل تحفيزية (streaks) وإحصاءات شخصية.

## المشكلة والحل

كثيرٌ منّا ينسى أذكار الصباح والمساء أو يفقد المتابعة. «أذكاري» يحوّل الوِرد إلى
عادة يومية عبر عدّاد لكل ذِكر، وحفظ تلقائي لتقدّمك، وسلسلة تحفيزية تنمو كل يوم.

## التقنيات

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4** — واجهة عربية RTL
- **Supabase** — قاعدة البيانات + المصادقة (Auth) + سياسات RLS

## الميزات

- 🔐 **تسجيل الدخول / إنشاء حساب** (Supabase Auth)
- 📿 **متتبّع الأذكار** — تبويب صباح/مساء، عدّاد لكل ذِكر، شريط تقدّم
- 👤 **الملف الشخصي** — السلسلة الحالية، أطول سلسلة، الأيام المكتملة، إجمالي الأذكار
- 💬 **الدعم والتواصل** — نموذج شكوى/اقتراح يُحفظ في قاعدة البيانات

## التشغيل محليًا

```bash
npm install
cp .env.local.example .env.local   # ثم املأ القيم من Supabase
npm run dev
```

### إعداد قاعدة البيانات

1. أنشئ مشروعًا على [supabase.com](https://supabase.com).
2. من **Project Settings → API** انسخ `URL` و `anon key` إلى `.env.local`.
3. افتح **SQL Editor**، الصق محتوى [`supabase/schema.sql`](supabase/schema.sql) واضغط **Run**.
4. (لتسهيل التجربة) من **Authentication → Providers → Email** يمكن إيقاف
   "Confirm email" مؤقتًا لتسجيل الدخول مباشرة بعد إنشاء الحساب.

## متغيّرات البيئة

| المتغيّر | الوصف |
| --- | --- |
| `NEXT_PUBLIC_SUPABASE_URL` | رابط مشروع Supabase |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | مفتاح anon / publishable |

## النشر على Vercel

اربط المستودع بـ Vercel، أضِف المتغيّرين أعلاه في **Settings → Environment
Variables**، ثم Deploy.
